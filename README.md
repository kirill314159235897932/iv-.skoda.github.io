## 🚗 О проекте

Простой и информативный одностраничный сайт (landing page) про популярный лифтбек Škoda Rapid. Сайт содержит:

- Основные характеристики и версии
- Описание двигателей (1.6 MPI, 1.4 TSI)
- Информацию о трансмиссиях (МКПП, АКПП Aisin, DSG)
- Габариты и клиренс
- Плюсы и минусы автомобиля
- Типичные проблемы на больших пробегах
- Советы по выбору на вторичном рынке
